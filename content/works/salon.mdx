## 悩んだ点
### 透明ヘッダー
ヘッダーを透明にしたため、スクロールすると背景イメージの色によってメニューが見ずらい問題が発生しまいた。
そのため、表示中のセクションによってヘッダーの背景色が変わるよう機能をContext APIで実装しました。
ここて問題になったのは、スクロールイベントにキャッチして、現在のセクションを判断することでした。
これは[**Intersection Observer API**](https://developer.mozilla.org/en-US/docs/Web/API/Intersection_Observer_API)を使って解決しました。

```js showLineNumbers
const io = new IntersectionObserver(entries => {
            entries.forEach((entry) => {
                if (entry.isIntersecting) { // 画面に表示されているセクションを判断
                    io.unobserve(entry.target); // 画面に表示されているセクションを監視対象から外す
                    const menuName = entry.target.id.replaceAll('-', ' ');
                    handleScrollPosition(menuName); // context apiのstateを更新
                }
            });
        },{ threshold: 0.2, rootMargin:'0px 0px -50% 0px'})
```
