# 1.プロジェクトの概要
旭川のリソース、スキー場の情報をまとめたアプリです。リアルタイム天気情報やスタンプラリーはもちろんレストランの検索も可能。

## 1.1 プロジェクト基本情報
- プロジェクト期間: **2020年10月 - 2021年4月**
- クライアント: 旭川観光コンベンション協会など
- チーム構成: 2人(APIは他社開発のため別当チーム構成)
- 役割: アプリ開発チーム員（各機能実装分担）

## 1.2 開発環境
1. Windows 10
2. MacOS(iOSビルドのみ利用)
3. Visual Studio Code
4. Bitbucket

## 1.3 使用言語/FW/ライブラリ
- React Native *(CLI)*
- Styled Components
- React Query

# 2. 開発案件
ログイン・セキュリティー対策などは他のエンジニアが担当し、私は主にユーザーのアクションによる各機能の実装を担当しました。
## 2.1 QRコードでのポイント購入
- ユーザーエクスペリエンスの向上を図るため、QRコードを使用してポイント購入機能を実装しました。
- ポイント購入画面詳しいデザイン指定がなかったためデザインも担当し、ロジックの実装を行いました。
## 2.2 リゾート・食堂検索（CMS連動）
- ユーザーがスキー場内のリゾート施設や食堂を検索し、詳細情報を閲覧できる機能を実装しました。
- 各ショップ・リゾート詳細ページには*carousel sliderを投入しましたが、React NativeのAnimated APIにパフォーマンスイシューがあり*、FPS低下が発生したため、**Reanimatedを使用してアニメーションを実装**しました。
## 2.3 スタンプラリー（GPS連動）
- スキー場内のアトラクションや特定のポイントでスタンプを集めるスタンプラリー機能を実装しました。
- ユーザーがアプリを使用してGPS情報をもとにスタンプの配置場所を探し、QRコードをスキャンしてスタンプを収集できる仕組みを構築しました。
## 2.4 天気情報・ライブカメラ連動
- ユーザーにとって重要な情報である天候情報とライブカメラ画像を提供するための機能を実装しました。
- サードパーティの天気情報APIとライブカメラストリームを統合し、ユーザーがリアルタイムの天候情報を確認できるようにしました。
- ユーザーはスキー場に行く前に天候や斜面の状況を把握でき、安全なスキーやスノーボード体験をサポートしました。

# 3. 成果
## 3.1 スキルの向上と学び
- プロジェクトを通じて、*React Native、Styled Components、React Queryなどの新しい技術やライブラリに触れ、* **そのスキルを磨く機会**となりました。
これにより、モバイルアプリケーションの開発において広範な知識と経験を得ることができました。特にReact NativeはExpoではなくCLIでの開発を行ったため、React Nativeの基礎知識を深めることができました。 
## 3.2 チームワークとコミュニケーション
- 社内では2人だけのチームでしたが、**他社とのコラボレーションプロジェクト**だったため、プロジェクトを成功に導くためのコミュニケーションスキルを見につけることができました。
APIの他社開発との連携や問題解決の過程で、円滑なコミュニケーションを図り、効果的な協力関係を築くことができました。
- 新人として初担当プロジェクトでしたが、協力社の中にはデザイン会社もあり、元デザイン専攻の私が**コミュニケーションをリードする経験**もできました。
## 3.3 ユーザーエクスペリエンスの設計
- ポイント購入、リゾート情報検索、スタンプラリー、天候情報の実装を通じて、ユーザーエクスペリエンスを重視する考え方を学びました。ユーザーの視点からアプリの機能を設計し、使いやすさを追求する方法についての洞察を深めました。
## 3.4 プロジェクトマネジメント
- プロジェクト期間内に複数の機能を実装し、スケジュールを守りながら成果物を提供するスキルを磨きました。タスクの優先順位付け、タイムマネジメント、リリース計画の策定など、プロジェクトマネジメントの重要性を理解しました。
